Mantid Imaging 3.1.0
====================

We are happy to announce the release of Mantid Imaging 3.1.0, which delivers a powerful set of features and improvements including an interactive 3D viewer, cone beam geometry support for tomographic reconstructions, Bragg edge fitting for neutron energy resolve imaging, and a host of other improvements. 

Key Highlights:
---------------

**Bragg Edge Analysis:**
Mantid Imaging 3.1.0 introduces the capability to perform Binned Bragg edge fitting analysis on energy-resolved neutron images, exporting the data to a table and fitted parameter overlay mapped onto the sample image. This powerful feature enables a range of scientific techniques for measuring structural and elemental properties across samples.

**3D Viewer:**
A dedicated 3D viewer has been added to Mantid Imaging which embeds the `CILViewer widget <https://github.com/TomographicImaging/CILViewer>`_ to allow for improved viewing of 3D samples, providing a level of control to toggle which planes are visible, slice visibility and volume render visibility.

**Geometry:**
A new dedicate window has been introduced into Mantid Imaging which allows for the creation and management of parallel and cone beam geometries for loaded datasets. Now that the geometry configuration is handled independently of the reconstruction window, users have greater control over geometry parameters.

**Operations:**
Two new operations have been added to Mantid Imaging which allow for the appending of image stacks or the summing of them. Summing allows for the emulation of increased exposure while appending allows various stacks which may be from the sample acquisition to be merged into one larger stack.

**Outreach**
Mantid Imaging now supports the exporting of a sub-sampled image stack as an animated GIF, making it easy to share and present data visually. It's a simple as right-clicking your sample in the main window and exporting as GIF. 

New Features
------------
- `#2136 <https://github.com/mantidproject/mantidimaging/issues/2136>`_: Mantid Imaging now supports cancelling a dataset on load
- `#2326 <https://github.com/mantidproject/mantidimaging/issues/2326>`_: Updated the code to display warnings for zero, negative, and NaN values using user-friendly colors in the operation and reconstruction windows.
- `#2347 <https://github.com/mantidproject/mantidimaging/issues/2347>`_: Support for Differently-Sized ROIs in Spectrum Normalisation.
- `#2395 <https://github.com/mantidproject/mantidimaging/issues/2395>`_: Adds an Image preview to the Export tab and makes the Table/Image view switchable from within the Export tab.
- `#2759 <https://github.com/mantidproject/mantidimaging/issues/2759>`_: Added a new Geometry window for editing, visualising, and creating new ImageStack Geometry data
- `#2766 <https://github.com/mantidproject/mantidimaging/issues/2766>`_: Implement backend logic and tests to enable ``Sum Stack Intensities`` operation which can sum tomography and time-of-flight datasets to emulate increased exposure.
- `#2783 <https://github.com/mantidproject/mantidimaging/issues/2783>`_: The initial guess for the ROI in the Cropping Operation now selects the important data in the image stack
- `#2800 <https://github.com/mantidproject/mantidimaging/issues/2800>`_: Updated the code so that the ROI created in the Operations window for Crop Coordinates, as well as the ROI set in the Main window, can now be resized from all four corners.
- `#2827 <https://github.com/mantidproject/mantidimaging/issues/2827>`_: Added support for 3D cone beam geometry
- `#2828 <https://github.com/mantidproject/mantidimaging/issues/2828>`_: The rois in the spectrum viewer now persist when the selected stack is changed.
- `#2844 <https://github.com/mantidproject/mantidimaging/issues/2844>`_: Fitting parameters in the Spectrum Viewer can now by bounded by a range or fixed number in the GUI.
- `#2848 <https://github.com/mantidproject/mantidimaging/issues/2848>`_: Added a dedicated 3D Viewer window in Mantid Imaging, providing the basis for embedding the CILViewer widget.
- `#2849 <https://github.com/mantidproject/mantidimaging/issues/2849>`_: Embedded the 3D viewer and stack selector widget in the viewer window.
- `#2874 <https://github.com/mantidproject/mantidimaging/issues/2874>`_: The user can now save sinogram data via Main Window->File->Save as Image Files and checking "Save as sinogram"
- `#2888 <https://github.com/mantidproject/mantidimaging/issues/2888>`_: Limit recon algorithm selection based on geometry
- `#2892 <https://github.com/mantidproject/mantidimaging/issues/2892>`_: The ROI Normalisation Operation now selects the largest air region available automatically.
- `#2902 <https://github.com/mantidproject/mantidimaging/issues/2902>`_: Updated fitting engine to return both Residual Sum of Squares (RSS) and Reduced RSS (RSS/DoF).
- `#2909 <https://github.com/mantidproject/mantidimaging/issues/2909>`_: When an angle csv file is loaded after the dataset, the dataset is ordered according to the angle data.
- `#2954 <https://github.com/mantidproject/mantidimaging/issues/2954>`_: Allow running fit on sub ROIs to update the export table
- `#2955 <https://github.com/mantidproject/mantidimaging/issues/2955>`_: Map fitted parameters over sample image, providing various control options for map visualisation and the ability to export mapped area only or mapping over sample image.
- `#2978 <https://github.com/mantidproject/mantidimaging/issues/2978>`_: The user can now right click a stack in the Main Window TreeView and rename it.
- `#2984 <https://github.com/mantidproject/mantidimaging/issues/2984>`_: Add a right-click context menu option in the main window image view to "Export stack as GIF animation".
- `#3008 <https://github.com/mantidproject/mantidimaging/issues/3008>`_: The Append Stack Operation is now functional. It allows the user to append two stacks together and sort them according to the projection angle data supplied in the associated log files.
- `#3015 <https://github.com/mantidproject/mantidimaging/issues/3015>`_: Add Cancel button and close confirmation to CoR/Iteration Refinement dialogs
- `#3016 <https://github.com/mantidproject/mantidimaging/issues/3016>`_: Add Delete Geometry button to Geometry Window
- `#3023 <https://github.com/mantidproject/mantidimaging/issues/3023>`_: Update main window angles after geometry creation
- `#3024 <https://github.com/mantidproject/mantidimaging/issues/3024>`_: Read COR and Tilt from geometry in recon window
- `#3038 <https://github.com/mantidproject/mantidimaging/issues/3038>`_: In the Main Window, the projection stack can be viewed as sinograms via the right click menu without modifying the ImageStack.
- `#3043 <https://github.com/mantidproject/mantidimaging/issues/3043>`_: Add the ability in the Spectrum Viewer to load in ROIs from a CSV file based on the same format that can be exported by the Spectrum viewer with basic format validation checks.
