MantidImaging#

Mantid Imaging is a graphical toolkit for performing 3D reconstruction of neutron tomography data. It provides an easy-to-use graphical interface to a wide range of pre/post-processing operations, tilt correction, and reconstruction algorithms, accommodating for tomography users with varying data complexity and image analysis background knowledge. It utilizes a flexible plugin system that allows easy integration of external software, and has allowed us to re-use software widely known in the neutron tomography community.

For more information, see the Overview.

Main Window

Main interface showing the newly loaded lego golden ratio dataset with autocolouring using Jenks classification and Viridis color mapping

Operations

Image processing operations panel displaying the remove outliers operation

Reconstruction

Tomographic reconstruction interface showing automatic center of rotation (COR) correction, taking the square sum of the projections as a noise heuristic to find the best COR for a given sinogram

Reconstruction Preview with Residuals and Convergence

Preview reconstruction using primal duel-hybrid gradient with total variance (PDHG-TV) and the visualisation of residuals and convergence metrics

Spectrum Viewer

The Spectrum Viewer with MCP collected Fe powder dataset loaded. The Spectrum Viewer allows for analysis of neutron time of flight (TOF) data enabling users to perform Bragg edge analysis

Geometry Correction

Cone beam tilt and rotation correction can be performed in the geometry window to produce high quality reconstructions
